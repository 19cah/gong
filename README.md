# Installation

```
$ gem install gong
```

# Usage

```
$ gong --help

Gong: gem to github

Usage
 $ gong <version>

 Version can be:
	 patch | minor | major 

Options
 --clean       Allow publishing from any branch
 --no-cleanup  Skips cleanup of git and ruby gems
 --tag         Publish under a given tag

Examples
 $ gong
 $ gong patch
 $ gong 2.3.0
 $ gong 1.0.3-beta.2 --tag=beta
```
